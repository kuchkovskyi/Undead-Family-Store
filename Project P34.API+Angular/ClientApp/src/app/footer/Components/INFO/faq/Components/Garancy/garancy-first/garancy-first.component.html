Які товари не підлягають обміну і поверненню?

Згідно зі статтею 9 Закону України «Про захист прав споживачів», Кабінетом Міністрів України затверджено перелік товарів належної якості, що не підлягають обміну або поверненню. Ось деякі з них:

  1. пір'яно-пухові вироби;

  2. рукавички;

  3. дитячі іграшки (м'які);

  4. білизна натільна;

  5. білизна постільна;

  6. панчішно-шкарпеткові вироби.
