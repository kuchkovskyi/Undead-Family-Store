Як змінити електронну адресу в аккаунті?

  1. Увійдіть до Особистого кабінету (посилання знаходиться у нижній частині сайту), ввівши логін і пароль.

  2. Виберіть "Редагувати профіль".

  3. Введіть нову адресу електронної пошти, натисніть "Зберегти".

  4. Натисніть "Змінити".

  5. На вашу нову пошту надійде лист-підтвердження.

  6. Підтвердіть нову електронну адресу. Після цього адресу пошти у вашому акаунті буде змінено.
