Як видалити аккаунт?

  1. Увійдіть до особистого кабінету, ввівши логін і пароль.

  2. Виберіть "Видалити профіль".

  3. За бажанням вкажіть причину видалення.

  4. Поставте галку про згоду з видаленням історії та іншими наслідками видалення облікового запису.

  5. Натисніть Видалити.

Ви можете видалити акаунт у будь-який час, коли забажаєте. Також ви можете створити новий акаунт на UNDEAD FAMILY.store.
