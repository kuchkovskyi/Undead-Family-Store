Як скасувати замовлення?

Якщо замовлення ще не оброблене

Після оформлення замовлення, воно обробляється на протязі трьох днів. Якщо на протязі цього часу Ви вирішили відхилити Ваше замовлення, спочатку перейдіть у розділ "Мої замовлення", виберіть замовлення, що Вас цікавить, та натисніть на кнопку "Відхилити".

Якщо замовлення вже оброблене

Після обробки Вашого замовлення, на Ваші номер телефону та електронну адресу прийде SMS-повідомлення з реквізитами для оплати. Надішліть зворотнє повідомлення та напишіть про те, що бажаєте відхилити замовлення.
