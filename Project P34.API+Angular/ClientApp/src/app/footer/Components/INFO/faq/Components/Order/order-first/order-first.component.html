Якщо оплачування відбулось через картку

Якщо ви оплатили замовлення карткою, на, вказаний Вами, номер телефону та електронну адресу прийде SMS-повідомлення з ТТН-кодом, який знадобиться Вам для того, щоб отримати товар на відділення Нової Пошти.

Якщо оплачування відбудеться накладеним платежем

Після обробки Вашого замовлення, на, вказаний Вами, номер телефону та електронну адресу прийде SMS-повідомлення з номером Вашого відправлення та реквізитами для оплати.
