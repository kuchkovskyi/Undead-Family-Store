Як використовувати промокод?

Покупка

Промокод надає Вам можливість придбати будь-який товар зі знижкою -10%, -25%, -30%, -45%. Щоб активувати промокод:

  1. Перейдіть на сторінку товару, що Вас зацікавив;
  2. Натисніть кнопку
    Активувати промокод ;
  3. У вспливаючому вікні введіть Ваш промо та натисніть на
    Активувати ;
  4. Після активації промокоду натисніть
    Додати у кошик .

Все ще потрібна допомога?
