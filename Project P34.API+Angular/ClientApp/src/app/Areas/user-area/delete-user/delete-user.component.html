Наслідки видалення акаунта

УВАГА! Видалений акаунт неможливо відновити.

Видаляючи свій акаунт, Ви усвідомлено погоджуєтесь з видаленням:

  1. свого списку бажань;

  2. своєї історії переглядів;

  3. своєї історії замовлень.

Також Ви усвідомлено відмовляєтеся від можливості отримувати листи про знижки та відповіді на Ваші коментарі.
